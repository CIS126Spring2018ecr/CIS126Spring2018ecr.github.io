What is Git?

Git is a tool. It is repositories for open source software development, on a large scope.

Who started Git?

Git was created by Linus Torvalds. The development began on April 3rd, 2005. It became self-hosting by April 7th.

What is Hub?

A Hub is a center of the network. For GitHub, it is what connects everyone. However, within GitHub there are certain commands. One is "git". "Hub" can be used to enchange the "git" commands.

Describe Repository and its role?

Repository is the digital directory. It is the storage space where the projects and code is saved and uploaded. Its role is to hold all the files, and all versions of those files (which includes the revision history).
